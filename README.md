# Feedback form using Nextjs and Google-Sheets API

## Tech Stack

- NextJS
- Chakra-UI
- React-Hook-Form
- googleapis

## Run Locally

Clone the project

```bash
  git clone https://github.com/GeoBrodas/nextjs-form-using-google-sheets-api.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
